# XML For You 使用教程

XML For You 是一个强大的在线 XML 编辑器和可视化工具箱，旨在最大限度地提升您处理和浏览 XML 数据的效率。它不仅支持基础的 XML 校验、格式化、压缩等功能，还提供 graph 和 table 两种功能丰富的可视化能力。本教程将详细介绍各项功能，帮助您快速上手，提升工作效率。

## 其他功能

XML For You 充分考虑用户的需求，还提供了以下功能：

- 拖拽上传：左右两侧编辑器都支持直接拖拽文件上传。

  <img src="/example/drag-drop.webp" alt="拖拽上传" style={{ marginBottom: 10 }} />

- 展示 XML path：光标停留在左侧编辑器上时，会在底部状态栏展示当前光标所在的 XML path。

  <img src="/example/json-path.webp" alt="展示 XML path" style={{ marginBottom: 10 }} />

- XML 解析错误提示：当 XML 格式错误时，会在底部状态栏展示错误提示，点击错误信息的话，光标会定位到编辑器中的错误位置。

  <img src="/example/validate-in-tutorial.webp" alt="校验 XML" style={{ marginBottom: 10 }} />
