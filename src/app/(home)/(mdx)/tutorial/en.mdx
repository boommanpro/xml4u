# XML For You Tutorial

XML For You is a powerful online XML editor and visualization toolbox designed to maximize your efficiency in processing and browsing XML data. It not only supports basic functions such as XML validation, formatting, and compression, but also provides two feature-rich visualization capabilities: graph and table. This tutorial will introduce each function in detail to help you get started quickly and improve your work efficiency.

## Other Features

XML For You also provides the following features to fully meet user needs:

- Drag and drop upload: Both the left and right editors support direct drag and drop file uploads.

  <img src="/example/drag-drop.webp" alt="Drag and drop upload" style={{ marginBottom: 10 }} />

- Show XML path: When the cursor is on the left editor, the XML path of the current cursor position will be displayed in the bottom status bar.

  <img src="/example/json-path.webp" alt="Show XML path" style={{ marginBottom: 10 }} />

- XML parsing error prompt: When the XML format is incorrect, an error prompt will be displayed in the bottom status bar. If you click the error message, the cursor will be located at the error position in the editor.
