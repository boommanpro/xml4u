## 项目介绍

本项目 fork [json4u](https://github.com/boommanpro/json4u)，在原项目基础上做了如下改动，作为自用

1. 删除登录功能，提高访问速度
2. 删除google统计，防止数据泄密
3. 增加url读取data入参功能，方便外部直接访问
4. 支持github build

## 使用方式

1. 直接点击编辑页面操作： [https://boommanpro.github.io/xml4u/editor](https://boommanpro.github.io/xml4u/editor)
2. 使用url进行传递： [https://boommanpro.github.io/xml4u/editor?data={"hello":"world"}](https://boommanpro.github.io/xml4u/editor?data=%7B%22hello%22%3A%22world%22%7D)
3. 现将内容copy到剪切板，然后点击链接： [https://boommanpro.github.io/xml4u/editor?data=copyFromClipboard](https://boommanpro.github.io/xml4u/editor?data=copyFromClipboard)
    可以结合raycast的快捷键使用。
